# ECO528 Term Paper: Black-Litterman Portfolio Optimization

## Overview
This repository contains the code implementation for the term paper "Bayesian Portfolio Selection via the Black-Litterman Model with Cross-Sectional Momentum Views".

## Requirements
- Python 3.8+
- Libraries listed in `requirements.txt`

## Installation
```bash
pip install -r requirements.txt
```

## Usage
To run the full pipeline, execute the main script:
```bash
python main.py
```
Alternatively, you can run the individual scripts sequentially from `01_download_data.py` to `07_analysis.py`.
